    • ABOUT ME

    My name is Mohammed. I was born and raised in Surulere, Nigeria. Since

    I was a kid I have always been faceted by computer, but unfortunately,

    I didn't have the resources to push my interest in these field.

    When I heard I'm going to be moving to the united states

    I was excited. I was finally going to peruse my interest in technology.

    As soon as i got into the United States. I started learning HTML and CSS

    online. I remember writing my first line of code I was the happiest person

    in the world. I ventured out into other programming language.

    I took Java and Python classes. I created a virtual assistant

    that respond to people question to with Python and Java. I felt

    something different when I started learning JavaScript. I started

    taking JavaScript courses online. I was accepted into google

    summer Internship. I was able to further my knowledge in JavaScript

    and solve real world problems. I have gained experience working

    in groups, writing clean code, Object oriented and algorithm for specific functions.

    I am still learning how to use modern framework like React, Bootstrap,

    and AOS. I currently working on how to use node JS to create backend servers.
